### LOC 기법
> 기간 산정 기법

- 핵심 키워드 
    - 총코드줄/노력 => 총코드줄/인원*기간

### 수학적 산정기법 종류
- COCOMO 모형
- Putman 모형
- 기능점수(Fountion Point)모형

#### COCOMO 기법
- Bohem 형이 제안한것으로 LOC에의한 산정 기법
- 비교적 작은규모를 통계 분석함
- 필요한 노력(Man-Month) -> (인원 기간)

#### COCOMO소프트웨어 개발 유형
> 코드라인 : 조직형(5만) < 반분리형(30만 이하) < 내장형(30만 이상)

- 조직형 : 중소규모
- 반분리형 : 트랜잭션 처리, DB 관리시스템
- 내장형 : 초대형 프로젝트 규모

#### PutMan 모형
- 노력의 `분포`를 가정해줌
- Rayleigh-Norden 곡선

#### FP - 가중치 증대 요인
- 자료입력
- 정보충력
- 명령어
- 데이터파일
- 필요한 외부 루틴과의 인터페이스

### 자동화 추정 도구 - `SLIM`
Rayleigh-Norden 곡선과 Putman 예측 모델을 기초로 하는 자동화 도구

### 네트워크 관련 신기술
> 총 13개 존재

- SD(Software Defined)시리즈 -> SDN(NetWork), SDS(Store), SDDC(Direct Connect)
- All Join : 오픈소스 IOT 플랫폼
- MeshNetWorkt : 대용량, 특수목적, Mesh로 안정적 그물망
- PICONET(피코넷) : 블루투스
- HSM(HardWare Security Manage) : 암호키, 생성 저장 처리를 클라우드로
- PaaS-TA(파스타)
- Zing : NFC
- WDN : 광섬유
- SSO : 한번의 로그인 모두 연결, Single Sign On
- Smart Grid : 전력선
- 메타버스

### SW관련 신기술 - 여기는 그냥 증발품 메시업만?

- 증발품 : 배포안함
- SOA

### 보안 관련 신기술
- BitLocker : 윈도우 7부터 지원, Window전용
- BaaS : 블록체인, 변경 용이
- OWASP : 비영리단체, 오픈 웹 애플리케이션 보안
- TCP Wrapper : 접속인가 여부 
- Honeypot : 당한척 하기

### HW 관련 신기술

- N-Screen : 여러대 기기 동시에

### DB관련 신기술

### 네트워크 구축
- 버스
- VLAN (Virtual LAN) : LAN의 물리적과 상관없이 논리적으로 연결
- WPA(WiFi Protected) : Wi-Fi에서 무선랜(WLAN) 비밀번호
- LAN의 표준 규격 - 802.11e : 매체접근제어(MAC)
- CSMA/CD-CA : CD | `충돌검출 기능` -> CA | `대기`

### (네트워크) 경로제어/트래픽제어

    - `거리백터`
    - `Bellman-Ford` 알고리즘
    - `최대 홉 수 15`
- OSPF(Open Shortest Path First Protocal)
    - `다익스트라` 알고리즘
    - 멀티 테스킹
- 흐름제어-정지-대기(`STOP-and-Wait`)
    - ACK(확인신호)
