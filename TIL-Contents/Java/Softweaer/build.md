# 빌드 자동화 도구
> 빌드란 소스코드 파일들을 여러개의 모듈을 `묶어` 실행 파일로 만듦,
이런 과정을 테스트 및 배포를 자동화 하는데 

이 <u>빌드 자동화 도구를</u> (Ant,Make,Meaven,Gradle,Jenkiens 등)이 있다.

### Jenkiens
> Java 기반의 오픈소스 형태로 Java에서 많이 사용되는 빌드 자동화 도구

- `서블릿 컨테이너`에서 실행되는 서버 기반 도구
- SVN,Git 등 대부분의 형상 관리 도구와 연동이 가능
- 친숙한 Web GUI 제공으로 사용이 쉽다. (문서 도움 화면)
-  여러대의 컴퓨터를 이용 분산 빌드나 테스트가 가능하다.

** 서블릿 컨테이너  **
>  클라이언트의 요청을 처리해주기 위해 서버측에서 실행되는 
`작은 프로그램`인 서블릿을 실행하고 생명주기를 관리하는 역할
